# minesweeper
Classic minesweeper clone with Swing GUI. Work in progress - BoardController unit tests are broken at the moment.

### Available difficulties (rows, columns, mines):
* beginner (9, 9, 10)
* intermediate (16, 16, 40)
* expert (16, 30, 99)
* pro (24, 40, 200)
* insane (50, 50, 625)
